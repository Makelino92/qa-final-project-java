## 👋 Bun venit! Ce este acest proiect?
Imaginează-ți că în loc să verifici tu cu mâna ta dacă un site funcționează, ai un **asistent virtual (un robot)** care face asta pentru tine în câteva secunde. Acest proiect este "creierul" acelui robot. 
- L-am scris în **Java** (limbajul de programare).
- Am folosit **Maven** (un fel de manager care are grijă ca robotul să aibă toate piesele necesare).

---

## 🛠️ Cum pornești robotul la tine pe calculator?

Dacă vrei să rulezi testele local, urmărește acești pași simpli:

### 1. "Ingrediente" necesare:
Pentru ca instrucțiunile să funcționeze, calculatorul tău are nevoie de:
* **Java (JDK):** Motorul care rulează codul.
* **Maven:** Asistentul care pornește testele. 
*(Dacă primești eroarea `'mvn' is not recognized`, înseamnă că Maven nu este instalat pe Windows-ul tău, dar nu-ți face griji, robotul de pe GitHub îl are deja pregătit!)*

### 2. Comanda de pornire:
Deschide o fereastră de comandă (CMD) în folderul proiectului și scrie:
```bash
mvn test
