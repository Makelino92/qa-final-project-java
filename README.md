## 👋 Ce face acest proiect?
Imaginează-ți că în loc să verifici tu manual dacă un site funcționează, ai un **robot** care face asta pentru tine. Acest proiect este "creierul" acelui robot. L-am scris în Java (limbajul de programare) și l-am organizat cu Maven (asistentul care pune totul în ordine).

## 🛠️ Cum pornești robotul la tine pe calculator?
Dacă vrei să vezi testele în acțiune pe computerul tău, e foarte simplu!

1. **Ustensile necesare:** Ai nevoie de Java și Maven instalate (sunt ca motorul și benzina pentru proiectul nostru).
2. **Comanda magică:** Deschide o fereastră de comandă (CMD) în acest folder și scrie:
   ```bash
   mvn test
